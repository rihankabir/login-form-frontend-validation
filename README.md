# Login Form Frontend Validation

A modern and responsive Login Form built using HTML, CSS, Bootstrap, JavaScript, and jQuery.

## Features

* Email validation
* Password validation
* Custom error messages
* Success message after validation
* Responsive design using Bootstrap
* Modern UI with glassmorphism effect
* jQuery form handling and validation
* User-friendly experience

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* jQuery
